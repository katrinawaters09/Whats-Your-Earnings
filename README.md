🧮 Payday Calculator (Python)
This is a custom-built Payday Calculator developed using Python and Tkinter. The application allows users to calculate their estimated earnings based on hourly rate, hours worked, and selected state tax rates.

🚀 Features
User-friendly calculator interface
Input hourly wage and hours worked
Automatic tax deduction based on selected state
Optional tax adjustments
Clear and backspace functionality
Keyboard and button input support
🛠️ Technologies Used
Python
Tkinter (GUI development)
📊 How It Works
The user enters:

Hourly rate
Hours worked
The calculator then:

Computes gross pay
Applies state tax deduction
Displays estimated net pay
